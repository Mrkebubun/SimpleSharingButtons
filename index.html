Simple Sharing Buttons Generator

Create light-weight, mobile-friendly sharing buttons for Facebook, Twitter, Google+ and other social media sites. Speed up your website or web app, avoid downloading unnecessary JavaScript files and keep your user's activity private. Choose from several styles, including Font Awesome -- completely free.

Download the icons

You can download Font Awesome from their official website. Alternatively you can include this code inside your head element:
<link href="http://netdna.bootstrapcdn.com/font-awesome/4.3.0/css/font-awesome.css" rel="stylesheet">
Note that you will also need to apply custom CSS (for color, hover style etc).

Don't forget to apply your own CSS (for color, hover style etc).

You can also download the sharing buttons code from the Github repo.
